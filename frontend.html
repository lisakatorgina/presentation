Технологии

JavaScript

— прототипно-ориентированный сценарный язык программирования. Является реализацией языка ECMAScript. JS поддерживаемый сейчас большинством браузеров, работает на ES5.

  • Транслирумые языки (CoffeScript, Babel)
  • Плагины и библиотеки (JQuery)
  • ECMAScript 6

Несмотря на то, что большинство разработчиков предпочитают работать с JS через библиотеки, такие как JQuery, необходимо понимать как работает язык и уметь пользваться его возможностями.

SVG

— Язык разметки масштабируемой векторной графики

  • хорошая браузерная поддержка (ie9+)
  • маленький вес изображений
  • может встаиваться инлайн
  • может быть стилизован с помощью css
  • поддерживает анимацию
  • существуют библиотеки для работы (raphael.js)
